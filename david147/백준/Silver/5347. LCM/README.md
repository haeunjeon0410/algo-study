### 문제 설명

<p>A fundamental step in adding and subtracting fractions is finding the least common multiple. Doing this by hand is difficult for some children (and some adults). You will write a program that finds the least common multiple of two positive integers. The least common multiple of two integers a and b is the smallest positive integer that is divisible by both a and b. </p>

### 입력 

 <p>The first line of input will be a positive integer, n, indicating the number of problem sets. Each problem set consist of two positive integers, separated by one or more spaces. There are no blank lines in the input.</p>

### 출력 

 <p>For each problem set print the least common multiple of the two positive integers. Print each least common multiple on a separate line.</p>
